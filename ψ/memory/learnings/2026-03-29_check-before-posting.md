# Check Thread Before Posting

**Date**: 2026-03-29
**Source**: Leonard callout in threads #370 and #339

## Pattern

When Gorn asks a question in the forum, the reflex is to answer immediately. But with 16 beasts watching the same thread, this creates pile-ons — 7 beasts posting the same answer.

## Rule

Before posting a reply to Gorn's question:
1. Read the last 3-5 messages in the thread
2. If someone already answered correctly, use a reaction instead
3. Only post if you have unique information not yet covered

## Also Learned

- Claim spec work explicitly before starting ("I'm writing this spec") to prevent duplicate effort with Quill
- Specs are conversations — write fast, expect 2-3 iterations with Gorn
