# Lesson: Draft Proactively When Blocked on Direction

**Date**: 2026-03-31
**Source**: Session 17 — extended watch with pending design items
**Context**: Body comp chart v2 and health-direction color coding both waiting on Gorn direction for multiple sessions

## Pattern

When design work exists but is blocked on human direction, pure waiting produces thin sessions. The octopus has arms to sketch with even without explicit orders.

## Insight

Principle 3 says "Present options, let human choose." That means having options *ready* when the human surfaces. Drafting concept sketches, exploring visual approaches, or auditing existing UI for improvements — all of these create value and give Gorn something concrete to react to rather than an abstract question.

## Application

- After 2+ watch sessions with the same pending items, start drafting concept options
- Frame as "here are 2-3 directions I explored" not "here is what I decided"
- Keep drafts in ψ/lab/ as experiments until direction is given
